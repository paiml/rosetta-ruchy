1. **Tuple Avoidance Pattern**
   ```ruchy

   fun get_second(data) -> T2 { ... }
   ```

2. **Vector-Based Data Structures**
   ```ruchy
   // Instead of: complex struct definitions
   // Use: parallel vectors with index relationships
   ```
